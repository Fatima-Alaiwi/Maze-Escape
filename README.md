# 🎮 Maze Escape

Maze Escape is a 3D First-Person game developed using Unity, where the player must navigate through a maze, collect gems, and reach the exit before time runs out.

---

## 🕹️ Gameplay

- Explore a maze environment
- Collect gems to progress
- Reach the exit before the timer reaches zero
- Avoid running out of time!

---

## 🎮 Controls

- **W A S D** → Move  
- **Mouse** → Look around  
- **Space** → Jump  

---

## ⏱️ Game Mechanics

- Countdown timer adds urgency
- Win condition: reach the exit
- Lose condition: time runs out
- Sound effects and fade transitions enhance gameplay

---

## 🛠️ Built With

- Unity (Game Engine)
- C# (Scripting)
- Starter Assets (First Person Controller)
